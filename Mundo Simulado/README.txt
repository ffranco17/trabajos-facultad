Este fue un proyecto hecho en grupo con otros compañeros de la materia Algortimos y Estructuras de Datos II. En el mismo se intenta modelar las reglas de un mundo simulado para el entrenamiento de agentes inteligentes (basado en los mundos desarrollados por OpenAI o XLand de DeepMind, en el siguiente link se puede obtener mas informacion sobre la idea original https://openai.com/blog/emergent-tool-use/).
En la carpeta src se encuentra el archivo con dicho código, asi como el código correspondiente al mapa de esta simulación y una implementacion de diccionarios de tipo Trie necesarios para la implementación de este programa.
